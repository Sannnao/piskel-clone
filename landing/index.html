Functionality overview:

    Tools:

  • Pen Tool

    Hot key: P

  • Mirror Pen Tool

    Hot key: V

  • Eraser Tool

    Hot key: E

  • Line Tool

    Hot key: L

  • Rectangle Tool

    Hot key: R

  • Fill Rectangle Tool

    Hot key: Ctrl + R

  • Circle Tool

    Hot key: C

  • Lighten Tool

    Hot key: U

  • Color picker Tool

    Hot key: O

    Interface:

  • Change tools size

    Hot key:
    • Plus size: ]
    • Minus size: [

  • Change fps

    Hot key:
    • Plus fps: F + "+" key
    • Minus fps: F + "-" key

  • Change Canvas size

    Hot key:
    • Plus size: Mouse wheel forward
    • Minus size: Mouse wheel backward

  • Swap color

    Hot key: X

    Frames

  • Add frame

    Hot key: N

  • Copy frame

    Hot key: Ctrl + N

  • Choose previous frame

    Hot key: Top arrow key

  • Choose next frame

    Hot key: Bottom arrow key
